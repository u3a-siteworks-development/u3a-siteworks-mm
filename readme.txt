Displays a maintenance page for site visitors when activated.

== Description ==

This plugin is part of the SiteWorks project.  When activated, it displays a maintenance message to visitors to the website.
A logged in Author, Editor or Administrator will still be able to access the website.

For more information please refer to the [SiteWorks website](https://siteworks.u3a.org.uk/)

== Frequently Asked Questions ==

Please refer to the documentation on the [SiteWorks website](https://siteworks.u3a.org.uk/u3a-siteworks-training/)
